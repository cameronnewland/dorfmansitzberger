Instructions

Custom portfolio page:

  1. Add a new page, assign the custom page template "Portfolio"
  2. Tell it which categories to use with a custom field named categories.
    Enter -0 to show all
    44 to show only category 44
    44,56 to show both category 44 and 56
  3. Optional: Override the default number of items per page by adding a custom field named posts_per_page.
    Enter a number of items to display per page or it will pull in your default WordPress settings.
  4. Optional: Hide the main page content of the portfolio page if you would like by adding a custom field of hide_content with any value such as yes - If you want a page title to appear, leave this setting alone and just enter <h2>Sample Heading</h2> in the content area set to HTML.
  5. Optional: Add a widget to the new widget area: Footer - Portfolio which is specific to this page.

This page is reusable so you are able to have multiple categories of items on your main menu

You can attach a post thumbnail to this page itself which will be sized to 940px and placed at the top.

Items displaying on this custom portfolio page need a post thumbnail image attached. Instructions on adding a post thumbnail are in the main instructions with this theme.

If you want to adjust the height of the thumbnails, edit this number in functions.php:
add_image_size( 'portfolio-thumb', 220, 220); // 4 column portfolio size
(You may need to delete and re-attach the post thumbnail for the change to take effect)

Alternate home page template:

  1. Add a new page, assign the custom page template "Home - No Slider"
  2. Add a widget to the new widget area: Home Slider Replacement which is specific to this page.
    Use the text widget will for embedding video or custom HTML
